# End-to-End ML Pipeline with MLflow

## Overview

This project focuses on building a reproducible Machine Learning workflow using MLflow.

The objective is to track experiments, manage model versions, and create a structured pipeline for training and evaluating machine learning models.

---

## Why MLflow?

Machine Learning projects often involve multiple experiments and model versions.

MLflow helps:

* Track experiments
* Log metrics
* Compare model performance
* Store model artifacts
* Improve reproducibility
